An Interaction Designer researches, designs and tests the scenarios, interaction patterns and interface systems that make up an interaction with the digital service. The interaction designer will make decisions based on the evidence of user needs in pursuit of the best service. The Interaction Designer should be skilled in using evidence-based approaches to create a user-centred and accessible service while creating or making use of appropriate visual design patterns. At a minimum, the Interaction Designer should possess enough familiarity with code to work closely with developers to make HTML prototypes. Ideally, they would be able to prototype collaboratively with other members of the multidisciplinary team.
